# Context Folder

Drop reference documents here before running build sessions. Claude reads these when building rule logic, parsers, and QC checks.

## What Goes Where

| Folder | Contents |
|---|---|
| `guidelines/uspap/` | Current USPAP edition (PDF or text extract) |
| `guidelines/gse/` | Fannie Mae B4-1, Freddie Mac 5600, FHA 4000.1 Ch.II.D, VA Ch.11 |
| `rule-references/uad-3.6/` | UAD Appendix D, MISMO XML schema XSD |
| `rule-references/gse-overlays/` | GSE-specific appraisal policy updates, SELs, bulletins |
| `sample-reports/1004/` | Redacted sample 1004 reports (PDF + XML) |
| `sample-reports/1073/` | Redacted sample 1073 condo reports |
| `sample-reports/1025/` | Redacted sample 1025 multi-family reports |
| `sample-reports/2055/` | Redacted sample 2055 exterior-only reports |

## Privacy Rule
NEVER add real appraisal reports with borrower names, SSNs, loan numbers, or unredacted property addresses. Use redacted or synthetic samples only.
